## 🚀 Getting Started

<div align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212284087-bbe7e430-757e-4901-90bf-4cd2ce3e1852.gif" width="400">
</div>

### 📋 Prerequisites

- ✅ Node.js 18+ (LTS)
- ✅ npm or yarn
- ✅ Google Gemini API key ([Get one here](https://ai.google.dev/))

### 📥 Installation

```bash
# Clone the repository
git clone https://github.com/Uthumnalinda/Agrigenius_AI.git

# Navigate to project directory
cd Agrigenius_AI

# Install dependencies
npm install
```

<p align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212257456-8efd59c4-74e8-46ea-8d08-0a18deac81f5.gif" width="200">
</p>

### ⚙️ Environment Setup

1. Create a `.env.local` file in the project root
2. Add your Google Gemini API key:

```bash
# .env.local
GEMINI_API_KEY=your_gemini_api_key_here
```

### ▶️ Running the App

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The app will be available at `http://localhost:5173` (default Vite port).

## 🧩 Usage

### 🌐 Using the Application

1. **Launch the app** using `npm run dev`
2. **Input agricultural data** through the user interface
3. **Get AI-powered insights** from Google Gemini
4. **View visualizations** and recommendations

### 🔧 API Integration

The app integrates with Google Gemini AI for intelligent analysis:

```typescript
import { GoogleGenerativeAI } from '@google/genai';

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);
const model = genAI.getGenerativeModel({ model: 'gemini-pro' });

// Example usage in components
const result = await model.generateContent('Analyze this crop data...');
```
